Smart Media Queries v1.1.0

    https://github.com/davidjcastner/smart-media-queries


To use CSS Classes:

    Include either smart-media-queries.css or smart-media-queries.min.css in
    your project and include the approiate link tag.

To use SCSS Mixins:

    Include _smart-media-queries.scss in your project and include the following
    at the top of your SCSS code:

        @import "/path/to/file/smart-media-queries";

To use both SCSS Mixins and CSS Classes:

    Include _smart-media-queries-classes.scss in your project and include the
    following at the top of your SCSS code:

        @import "/path/to/file/smart-media-queries-classes";
